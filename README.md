# Text Edit & Paste

This is a Raycast extension that allows you to edit and paste text directly to the active window.

## Features

- Edit text in a text area
- Press `Cmd+Enter` to paste text directly to the active window
- Save and manage text snippets
- Load and use saved snippets
- Delete unwanted snippets

## Usage

### Text Editing and Pasting
1. Enter or edit your text in the text area
2. Press `Cmd+Enter` to paste the text directly to the active window

### Saving and Managing Snippets
1. Enter the text you want to save in the text area
2. Enter a name for the snippet in the "Snippet Name" field
3. Click the "Save Snippet" button
4. To switch to the snippet management view, click the "View Snippets" button in the action panel

### Using Saved Snippets
1. Click "View Snippets" to switch to snippet list view
2. Select a snippet from the list
3. Click "Use This Snippet" to load the snippet content back to the text editor
4. Edit as needed and press `Cmd+Enter` to paste

### Managing Snippets
In the snippet list view:
- Click "Use This Snippet" to load the content to the editor
- Click "Delete Snippet" to remove a snippet
- Use the search bar to filter snippets by name or content
- Click "Create New" to return to the text editing view
