# Text Edit & Paste

Edit and paste the text

## Features

- Edit text in a text area
- Press `Cmd+Enter` to paste text directly to the active window
- Save and manage text snippets
- Load and use saved snippets
- Delete unwanted snippets

## Usage

Edit text in the text area and press `Cmd+Enter` to paste it directly to the active window.

You can also save frequently used text as snippets for later use. To save a snippet, enter the text and a name, then click the "Save Snippet" button. You can view and manage your saved snippets by switching to the snippet list view.
